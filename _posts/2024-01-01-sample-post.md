# Welcome to Your Blog

This is a sample blog post. You can write your thoughts, tutorials, or anything you'd like to share here.

## Markdown Support

You can use all standard Markdown features:

- Bullet points
- **Bold text**
- *Italic text*
- [Links](https://example.com)
- `Code snippets`

### Code Blocks

```python
def hello_world():
    print("Hello, World!")
```

### Images

You can also add images:

```markdown
![Alt text](/assets/images/your-image.jpg)
```

## Getting Started

To create a new blog post:

1. Create a new file in the `_posts` directory
2. Name it with the format: `YYYY-MM-DD-title.md`
3. Add front matter (the content between the `---` lines at the top)
4. Write your content in Markdown

Happy blogging!
